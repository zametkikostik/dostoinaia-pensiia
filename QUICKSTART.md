# 🚀 Быстрый старт — Пенсионный портфель

## Для пользователя (готовое APK)

### Шаг 1: Получить токен Т-Инвестиции

1. Открыть приложение **Т-Инвестиции**
2. Перейти в **Настройки** → **API**
3. Нажать **Создать токен**
4. Выбрать права:
   - ✅ `read` — чтение портфеля
   - ✅ `marketdata` — котировки
   - ❌ `trade` — **НЕ нужно**
5. **Скопировать токен** (показывается один раз!)

---

### Шаг 2: Настроить Cloudflare Worker

1. **Создать Worker:**
   - Зайти в https://dash.cloudflare.com/
   - Workers & Pages → Create Application → Create Worker
   - Назвать: `pension-portfolio-worker`
   - Нажать **Deploy**

2. **Вставить код:**
   - Открыть Worker → Code
   - Скопировать код из `cloudflare-worker/src/worker.js`
   - Вставить в редактор
   - Нажать **Save and Deploy**

3. **Создать KV Namespace:**
   - Workers & Pages → KV → Create namespace
   - Назвать: `PUSH_STORE`
   - Скопировать ID

4. **Добавить binding:**
   - Worker → Settings → Bindings → Add → KV Namespace
   - Variable: `PUSH_STORE`
   - Namespace: выбрать созданный
   - Save

5. **Добавить токен Tinkoff:**
   - Settings → Variables → Environment Variables
   - Add variable: `TINKOFF_TOKEN` = ваш токен
   - Save and Deploy

---

### Шаг 3: Настроить APK

1. **Открыть `env.local`:**
   ```bash
   # Путь к файлу
   /home/kostik/Пенсионный портфель/env.local
   ```

2. **Заполнить данные:**
   ```
   TINKOFF_TOKEN=ваш_токен_из_шага_1
   WORKER_URL=https://pension-portfolio-worker.kostik.workers.dev
   ```

3. **Сохранить файл**

---

### Шаг 4: Собрать APK

```bash
# Перейти в папку проекта
cd "/home/kostik/Пенсионный портфель"

# Запустить скрипт сборки
./build-apk.sh
```

**APK будет в папке:**
```
/home/kostik/Пенсионный портфель/releases/
  ├── app-debug.apk      (для тестирования)
  └── app-release.apk    (для установки)
```

---

### Шаг 5: Установить APK на телефон

1. **Передать APK на телефон:**
   - По USB
   - Через email/мессенджер
   - Через облако (Google Drive, Dropbox)

2. **Разрешить установку:**
   - Настройки → Безопасность → Неизвестные источники
   - Разрешить для браузера/файлового менеджера

3. **Установить:**
   - Открыть APK файл
   - Нажать "Установить"

---

### Шаг 6: Настроить приложение

1. **Открыть приложение**

2. **Ввести токен:**
   - Нажать ⚙️ (настройки)
   - В поле "Токен Т-Инвестиции" вставить токен
   - Нажать "Сохранить"

3. **Проверить:**
   - Вернуться на главный экран
   - Потянуть вниз для обновления
   - Должны загрузиться данные портфеля

## ❓ Частые вопросы

### Q: Как обновить APK?
**A:** Собрать новую версию тем же ключом:
```bash
./build-apk.sh
```
Установить поверх старой версии.

### Q: Можно ли использовать другой токен?
**A:** Да, в настройках приложения можно изменить токен.

### Q: Как часто делать ребалансировку?
**A:** Рекомендуется раз в квартал или при отклонении >5%.

### Q: Что если Worker не отвечает?
**A:** Проверить:
1. Worker задеплоен
2. Токен Tinkoff добавлен в переменные
3. KV namespace подключён
