# Пенсионный портфель 📈

Сбалансированный микс акций и облигаций для стабильного пассивного дохода и независимости от государства.

## 🎯 О проекте

Мобильное приложение для управления инвестиционным портфелем с подключением к **Т-Инвестиции API**.
Полуавтоматическая ребалансировка — приложение рекомендует, вы решаете.

### Возможности

- ✅ Подключение к Т-Инвестиции API (gRPC/REST)
- ✅ Синхронизация с реальным брокерским счётом
- ✅ Отслеживание котировок акций и облигаций
- ✅ Расчёт дивидендной доходности и купонов
- ✅ **Полуавтоматическая ребалансировка** — рекомендации по покупке/продаже
- ✅ Прогнозирование пассивного дохода
- ✅ Дивидендный календарь и купонные выплаты
- ✅ Push-уведомления о важных событиях
- ✅ Визуализация структуры портфеля

## 🚀 Быстрый старт

### 1. Настройка Cloudflare Worker

#### 1.1. Создать Worker

1. Зайти в [Cloudflare Dashboard](https://dash.cloudflare.com/)
2. Войти под своим аккаунтом
3. В левом меню выбрать **Workers & Pages**
4. Нажать **Create Application** → **Create Worker**
5. Ввести имя: `pension-portfolio-worker`
6. Нажать **Deploy**

#### 1.2. Вставить код Worker

1. Открыть созданный Worker → **Quick Edit**
2. Скопировать содержимое файла `cloudflare-worker/src/worker.js`
3. Вставить в редактор Cloudflare
4. Нажать **Save and Deploy**

#### 1.3. Создать KV Namespace

1. Workers & Pages → **KV** → **Create a namespace**
2. Ввести имя: `PUSH_STORE`
3. Нажать **Add**

#### 1.4. Добавить KV Binding

1. Worker → **Settings** → **Bindings** → **Add** → **KV Namespace**
2. Variable name: `PUSH_STORE`
3. KV namespace: выбрать `PUSH_STORE`
4. **Save**

#### 1.5. Добавить токен Т-Инвестиции

1. Worker → **Settings** → **Variables** → **Add variable**
2. Variable name: `TINKOFF_TOKEN`
3. Value: ваш токен из Т-Инвестиции
4. ✅ Включить **Encrypt**
5. **Save and Deploy**

---

### 2. Получение токена Т-Инвестиции

1. Т-Инвестиции → **Настройки** → **API** → **Создать токен**
2. Выбрать права: `read`, `marketdata`
3. Скопировать токен

---

### 3. Установка APK

1. Скачать `releases/app-release.apk`
2. Передать на телефон
3. Установить (разрешить установку из неизвестных источников)
4. В приложении ввести токен Т-Инвестиции

---

### 4. Сборка APK (для разработчиков)

```bash
# Склонировать репозиторий
git clone https://github.com/zametkikostik/dostoinaia-pensiia.git

# Настроить env.local (скопировать из env.local.example)
cp env.local.example env.local

# Запустить сборку
./build-apk.sh

# APK будет в папке releases/
```

## 📊 Рекомендуемая структура портфеля

| Категория | Доля | Инструменты | Надёжность |
|-----------|------|-------------|------------|
| Голубые фишки | 25% | Сбербанк, Газпром, Лукойл | ⭐⭐⭐ |
| Дивидендные акции | 25% | МТС, Норникль, Татнефть, Северсталь-Д | ⭐⭐⭐ |
| Облигации ОФЗ | 45% | ОФЗ-ПД 26234, 26238, 26244, 29024 | ⭐⭐⭐ (гос.) |
| Золото | 5% | Золото (GLDRUB_TOM) | ⭐⭐⭐ |

> ✅ **Только надёжные инструменты:** Никаких корпоративных облигаций с высоким риском. 
> Только государственные ОФЗ — минимальный риск дефолта.

## 🔁 Ребалансировка

**Полуавтоматический режим:**
- Приложение рассчитывает отклонения от целевого распределения
- При отклонении >5% — рекомендация к покупке/продаже
- Вы принимаете решение и выполняете сделки в Т-Инвестиции

## 🔐 Безопасность

- Токен хранится в зашифрованном виде (Android Keystore)
- Все запросы по HTTPS
- Возможность блокировки приложения (PIN/биометрия)
- Токен с правами только на чтение (`read`, `marketdata`)

## 📡 API

### Tinkoff Invest API

Документация: https://tinkoff.github.io/investAPI/

Используемые методы:
- `UsersService.GetAccounts` — счета клиента
- `MarketDataService.GetLastPrices` — последние цены
- `OperationsService.GetPortfolio` — портфель по счёту
- `OperationsService.GetPositions` — открытые позиции
- `InstrumentsService.GetShares/Bonds` — список инструментов

- **Backend**: Cloudflare Workers (JavaScript)
- **Storage**: Cloudflare KV, Room Database
- **API**: Tinkoff Invest API v2
